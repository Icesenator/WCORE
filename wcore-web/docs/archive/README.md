# WCORE Documentation Archive

This folder contains historical documents that are useful for context but are no longer the source of truth for the current project state.

Current sources of truth:

- `ROADMAP.md` — current state, consolidated TODO, product/technical direction.
- `CHANGELOG.md` — release-level changes.
- `README.md` — project overview and setup.
- `DEPLOY.md` — deployment notes and operational checklist.
- `AGENTS.md` — agent/operator gotchas and project-specific rules.

Archived files keep their original content so no audit or planning information is lost.

Archived root documents:

- `AUDIT.md` — historical 2026-05-11 audit snapshot.
- `SESSION_SUMMARY.md` — historical v0.1.18 handoff.
- `RELEASE_NOTES.md` — historical release notes superseded by `CHANGELOG.md` and `ROADMAP.md`.

Archived prompts:

- `gpt5.5/*.md` — historical prompt/response material used during early audit and backlog work.

Before using an archived action item, check `ROADMAP.md` first. Many findings are already closed or superseded.
